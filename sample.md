# Welcome to Markdown Reader

This is a **sample file** so you can test the app immediately after building it.

## Features

- Comfortable, book-like typography
- Dark theme that's easy on the eyes
- Adjustable font size (use the *A- / A+* buttons)
- Drag & drop support
